# Emergency

Critical disease alert documents used by the Rule-Based Emergency Advisory Module.

Each document in this directory describes a high-priority condition (e.g., Newcastle disease outbreak, Avian Influenza suspicion) and provides deterministic first-response guidance that does not depend on LLM availability.

## Schema

Documents here follow the standard knowledge base schema defined in `docs/knowledge_base_schema.md` with `domain: emergency` and must include an `## Emergency Response` section.
