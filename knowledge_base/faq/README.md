# FAQ

Frequently asked questions from poultry farmers, curated for high-recall retrieval.

FAQ documents are short, question-answer pairs that cover common practical queries not addressed in depth by the main domain documents. They are optimised for direct retrieval and short-context responses.

## Schema

Documents here follow the standard knowledge base schema defined in `docs/knowledge_base_schema.md` with `domain: faq`.
