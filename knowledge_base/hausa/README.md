# Hausa

Hausa-language knowledge base documents for northern Nigerian and Niger poultry farmers.

This directory mirrors the structure of the English knowledge base. Documents are translated or natively authored in Hausa and follow the same schema defined in `docs/knowledge_base_schema.md` with an additional `language: ha` metadata field.

## Status

Planned for Sprint 6 (Multilingual Support). This directory is created in Sprint 1.5 to reserve the namespace and signal architectural intent.

## Schema Extension

Hausa documents include one additional required metadata field:

```yaml
language: ha
```
