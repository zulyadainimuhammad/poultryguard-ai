# References

Bibliographic and source reference documents for the PoultryGuard AI knowledge base.

This directory contains structured reference lists, source attributions, and external resource guides that support the factual claims made across all knowledge base domains. References are not indexed for RAG retrieval but are used by the knowledge base validator to verify source attribution.

## Purpose

- Provide traceability from knowledge base claims to authoritative sources
- Support ADTC submission evidence requirements
- Enable domain expert review and fact-checking
